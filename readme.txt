Logo:
- Ersetzen Sie `images/logo.svg` durch Ihr eigenes Logo (gleicher Dateiname), dann ist es automatisch überall eingebunden.

Deployment (GitHub Pages):
1) Erstellen Sie ein neues Repo (z. B. `skycarrent`).
2) Laden Sie den Inhalt des ZIPs in das Repo (Root-Ebene).
3) In den Repository-Einstellungen GitHub Pages aktivieren: Source = `main` / Root.
4) Nach wenigen Minuten ist die Seite online.
